# 投资人核心策略

## 三层决策框架（2026-07-15 投资人确认）

### 第一层：大盘状态 → 决定总仓位

| 大盘状态 | 判断依据 | 目标仓位 |
|----------|----------|----------|
| **上行** | 突破压力位、均线向上 | 重仓（5成以上） |
| **震荡** | 区间波动、方向不明 | 中等（3-5成） |
| **下行** | 跌破支撑位、均线向下 | 轻仓（1-3成） |

### 第二层：个股类型 → 决定个股操作

| 个股类型 | 定义 | 操作 |
|----------|------|------|
| **趋势股** | 长期看好、行业赛道好、龙头强者恒强 | 短期回调可保留，轻仓观望 |
| **短线股** | 吃短线红利、非长期持有 | 该清就清，不恋战 |
| **刚买入就跌破止损** | 判断错误或入场时机有误 | 果断清仓，等企稳再考虑 |

### 第三层：止损执行 → 具体操作

- **趋势股**止损 → 保留30%轻仓（相信长期逻辑，等待企稳）
- **非趋势股**止损 → 清仓（逻辑不对，及时止损）

## 关键规则

1. **总仓位控制**：大盘状态决定目标仓位，个股灵活调整
2. **短期回调 vs 趋势走坏**：
   - 短期回调：长期逻辑没变，短期波动 → 趋势股保留（跌了还会涨回去）
   - 趋势走坏：长期逻辑被破坏 → 清仓
   - 判断标准：长期逻辑还在不在？（如AI趋势→内存稀缺→受益，跌了也会涨）
3. **大盘好 ≠ 个股好**：大盘上行不代表每只票都涨，个股该止损就止损
4. **不冲突**：总仓位1-3成（大盘下行时）≠ 个股必须留底仓。个股该卖就卖，总仓位由新开仓控制

## 趋势股的三维度判断（代码实现）

1. **技术趋势**：`highestPrice > costPrice * 1.02`（曾>2%浮盈）
2. **赛道趋势**：属于未来趋势行业（新能源、半导体、AI等17个行业）
3. **龙头长期看好**：`longTermBullish = true`（买入时标记）

任一满足即视为趋势股 → 止损时保留30%
