# 投资人核心策略

## 仓位管理原则（2026-07-15 确认，2026-07-15 修正）

**核心：1-3成指总仓位，个股止损该清就清**

| 市场趋势 | 总仓位要求 | 操作策略 |
|----------|-----------|----------|
| 趋势向下 | 1-3成（轻仓） | 总仓位控制在轻仓，但不代表每只票都留底仓 |
| 趋势企稳 | 逐步加仓 | 观察趋势站稳后开始加仓 |
| 趋势向上 | 重仓 | 顺势加仓，最大化收益 |

## 关键规则
1. **1-3成指总仓位**：不是每只个股都要留底仓，而是整体仓位控制在1-3成
2. **趋势股 vs 止损清仓**：
   - 趋势股（长期看涨）+ 已有浮盈 + 短期回调 → 轻仓保留，不全清，长期趋势没破
   - 刚买入就跌破止损 → 果断清仓，说明判断或入场时机有误，等企稳再考虑
3. **大盘好 ≠ 个股好**：大盘趋势向上不代表每只票都涨，个股该止损就止损
4. **趋势走坏 → 赶紧轻仓**：无论个股趋势向下还是大盘向下，第一时间减到轻仓，不犹豫
5. **不冲突**：总仓位1-3成 ≠ 个股不清仓。个股该卖就卖，总仓位由新开仓控制

## 与模拟盘的关系
- 模拟盘（stock-strategy）每日交易脚本应遵循此策略
- 实盘（东方财富账户）也按此原则操作
